создание директории:  		   mkdir Telegram_Chat_Scraper
переход в директорию: 		   cd Telegram_Chat_Scraper

создание виртуального окружения:   python3 -m venv venv
активация виртуального окружения:  source venv/bin/activate
Устанавливаем зависимости c файла: pip install -r requirements.txt
